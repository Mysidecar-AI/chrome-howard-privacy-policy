Battlecard Chrome Extension Privacy Policy

Last updated: June 11, 2025

The Battlecard Chrome extension, developed by Mysidecar, generates competitive analysis reports by comparing two website URLs. This Privacy Policy outlines our minimal data usage.

Data We Collect

  • Website URLs: The URL of the active browser tab (via Chrome’s tabs API) and a user-entered company URL.
  • No personally identifiable information (e.g., name, email, or location) is collected.

How We Use Data

  • URLs are sent to our secure API to generate battlecard reports.
  • Results are stored locally on your device using Chrome’s storage API for session persistence.

Data Sharing

We do not share, sell, or transfer data to third parties, except for API processing to fulfill the extension’s purpose.

Data Security

Data is securely transmitted and stored locally. We implement reasonable safeguards, but users assume submission risks.
